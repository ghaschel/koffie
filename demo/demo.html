.colored-text-border();

Example

.colored-text-border();
This may be used as a outline text.

Usage

.colored-text-border(color-1, size-1 (in px), color-2, size-2 (in px), color-3, size-3 (in px), color-4, size-4 (in px));

.colored-text-border(); - This will use the default sizes and colors.

.colored-text-border(yellow, 2, green, 2, red, 2, blue, 2);

.colored-text-border(#ffff00, 2, #008000, 2, #ff0000, 2, #0000ff, 2);

.colored-text-border(rgba(0, 128, 0, 0.98), 2, rgba(255, 255, 0, 0.98), 2, rgba(255, 0, 0, 0.96), 2, rgba(0, 0, 255, 0.91), 2);

Output

text-shadow: 2px 2px #ffff00,
            -2px -2px #008000,
             2px -2px #ff0000,
            -2px 2px #0000ff;

.error-input();

Example

It generates an error-explicit class for the inputs. Useful for field validation.
Both text and placeholder are colored.

Usage

.error {
    .error-input();
    /* You can use colors inside the parents */
    /* RGBA/RGB/Hex/Nominal colors */
}

Output

.error {
    border: 1px solid #e36d56 !important;
    color: #e36d56;
}
.error::-webkit-input-placeholder{
    color: #e36d56;
}
.error::-moz-placeholder {
    color: #e36d56;
}
.error:-moz-placeholder {
    color: #e36d56;
}
.error:-ms-input-placeholder {
    color: #e36d56;
}

.no-outline();

Example

Removes focus state outline.

Usage

input {
    &.wanted-class {
        .no-outline();
    }
}

Output

input.wanted-class {
    outline: none;
}
input.wanted-class:focus {
    outline: none;
    /*Firefox still shows outline, if we not declared on the focus state, so this fix it*/
}

.antialiasing;
.anti-aliasing;
.font-smoothing;

Example

Antialiased Text.
Enables Webfont smoothing, and antialiasing when available.

    -ms-text-rendering: optimizeLegibility;
    -o-text-rendering: optimizeLegibility;

    -moz-font-smoothing: antialiased;
    -ms-font-smoothing: antialiased;
    -o-font-smoothing: antialiased;
    font-smoothing: antialiased;
}

.top-triangle();

Example

Tooltip Triangle
Displays a triangle tooltip.
Must be used inside an after or before pseudoelement.

Usage

div {
    &:before {
        .top-triangle(15px, #ccc);
    }
}
/* or */
div {
    &:after {
        .top-triangle(15px, #ccc);
    }
}

Output

/* the border sizes and the pseudo element position is based on the size inserted in the mixin call*/
div:before {
    content: '';
    display: inline-block;
    height: 1px;
    width: 1px;
    position: absolute;
    top: -29px;
    border-left: 15px solid transparent;
    border-top: 15px solid transparent;
    border-bottom: 15px solid #cccccc;
    border-right: 15px solid transparent;
}

Example

Inventore recusandae soluta molestiae illum accusantium enim debitis veniam vero repellendus, dignissimos voluptatum ducimus ea minima sed maxime earum quam eaque rem.Lorem ipsum dolor sit amet, consectetur adipisicing elit. Ut reiciendis iusto omnis dolores quisquam architecto nobis repudiandae non? Eaque, beatae.
Changes the default scroll bar color.
NOTE: This only works in webkit, for now.

Usage

.class {
    height: 200px;
    overflow-y: scroll;
    .scroll-bar(6px, #ccc, darkblue, 0px);
}
/* Parameters:
    1) Width
    2) Background-color
    3) Thumb color
    4) Thumb Radius */

Output

.class::-webkit-scrollbar {
    width: 6px;
    height: 6px;
}
.class::-webkit-scrollbar-track {
        background: #ccc;
}
.class::-webkit-scrollbar-thumb {
        background: darkblue;
        -webkit-border-radius: 0px;
        border-radius: 0px;
}
